# Which Mountain To Climb

- 開發：一個對SwiftUI的簡單小嘗試
- 問題：與朋友相揪爬山，卻不知道要爬哪座山時，可以打開這個App抽山
- 使用：用戶進入App後，首先選擇想要爬的難度，再點我「點我抽抽」，即可抽山
